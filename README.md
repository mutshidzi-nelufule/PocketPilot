# PocketPilot

PocketPilot is an Android expense tracking application developed using Kotlin and Room Database.

## Features
- Add, edit, and delete expenses
- View expenses using RecyclerView
- Filter expenses by date
- Calculate category totals
- Display data using charts
- Attach images to expenses

## Technologies Used
- Kotlin
- Android Studio
- Room Database
- RecyclerView
- Coroutines
- MPAndroidChart

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Build and run the app
